Aula SASS: Mapas

No SASS, mapas são estruturas de chave e valor que permitem organizar dados em pares. Eles são úteis para centralizar configurações de estilos, como cores, tamanhos ou qualquer outro conjunto de valores que você queira reutilizar de forma dinâmica.

1. Criando um Mapa

Você pode criar um mapa utilizando a sintaxe de chave e valor:


// SCSS
$cores: (
  "primaria": #007bff,
  "sucesso": #28a745,
  "perigo": #dc3545,
  "aviso": #ffc107
);

2. Iterando Sobre Mapas

Você pode usar @each para iterar sobre as chaves e valores de um mapa:


// SCSS
@each $nome, $cor in $cores {
  .alerta-#{$nome} {
    background-color: $cor;
    color: #fff;
    padding: 10px;
    margin-bottom: 10px;
  }
}

Código SCSS Completo


// SCSS
$cores: (
  "primaria": #007bff,
  "sucesso": #28a745,
  "perigo": #dc3545,
  "aviso": #ffc107
);

@each $nome, $cor in $cores {
  .alerta-#{$nome} {
    background-color: $cor;
    color: #fff;
    padding: 10px;
    margin-bottom: 10px;
    border-radius: 5px;
  }
}

Conclusão

Os mapas no SASS permitem organizar e acessar dados de forma simples e eficiente. Use-os para centralizar suas configurações e gerar estilos dinâmicos com facilidade!
