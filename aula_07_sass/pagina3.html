Aula SASS: Condições e Laços

O SASS suporta o uso de condições (@if, @else, @else if) e laços (@for, @each, @while) para criar estilos dinâmicos. Estas funcionalidades permitem escrever CSS mais inteligente e flexível.

1. Condições (@if, @else)

As condições permitem aplicar estilos diferentes com base em variáveis ou valores.

Exemplo


// SCSS
$tema: dark;

body {
  @if $tema == dark {
    background-color: #333;
    color: #fff;
  } @else {
    background-color: #fff;
    color: #333;
  }
}

2. Laços (@for, @each, @while)

Os laços permitem gerar repetidamente estilos baseados em uma sequência ou coleção de valores.

Exemplo com @for


// SCSS
@for $i from 1 through 5 {
  .btn-#{$i} {
    padding: $i * 5px;
    font-size: $i * 2px + 10px;
  }
}

No exemplo acima, estamos gerando cinco variações de botões com tamanhos diferentes.

Exemplo com @each


// SCSS
$cores: red, blue, green;

@each $cor in $cores {
  .alerta-#{$cor} {
    background-color: $cor;
    color: #fff;
    padding: 10px;
  }
}

Exemplo com @while


// SCSS
$i: 1;

@while $i <= 3 {
  .coluna-#{$i} {
    width: $i * 10%;
  }
  $i: $i + 1;
}

Conclusão

Com condições e laços, você pode escrever estilos dinâmicos, reduzir repetição de código e manter seu CSS mais organizado e eficiente. Teste os exemplos e aplique-os em seus próprios projetos!
