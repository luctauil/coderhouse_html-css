Aula SASS

O SASS é uma extensão poderosa do CSS que adiciona funcionalidades com variáveis tornando-o mais eficiente e fácil de manter. A seguir, vamos aprender como instalá-lo e utilizá-lo em seus projetos.

1. Pré-requisitos

Antes de começar, é necessário que o Node.js esteja instalado no seu computador. Acesse o site oficial do Node.js para fazer o download e instalação: https://nodejs.org/en/

Após a instalação, verifique se o Node.js e o npm estão funcionando corretamente usando os comandos:

node -v
npm -v

2. Instalando o SASS

Você pode instalar o SASS de duas formas:

2.1. Instalação Global

Para instalar o SASS globalmente, execute o seguinte comando no terminal:

npm install -g sass

Com essa instalação, você poderá utilizar o comando sass em qualquer projeto diretamente no terminal.

2.2. Instalação Local (Projeto Específico)

Para instalar o SASS em um projeto específico, navegue até o diretório do seu projeto e execute:

npm install sass --save-dev

Isso adicionará o SASS como dependência de desenvolvimento no arquivo package.json do projeto.

2.3. Executar o SASS (Dentro do seu projeto)

Para para executar basta rodar o comando:

sass arquivo_sass.scss arquivo_saida.css

Esse comando vai gerar um arquivo com nome arquivo_saida.css para você configurar no seu projeto.

Conclusão

Agora você está pronto para usar o SASS em seus projetos! Escolha a forma de instalação que melhor se adapta ao seu fluxo de trabalho e aproveite todas as funcionalidades que o SASS oferece para otimizar o seu CSS.
